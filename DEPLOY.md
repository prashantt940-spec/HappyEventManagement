Happy Event Management Website

A static website for Happy Event Management ready for free hosting on Netlify and GitHub Pages.

## Quick Deploy

### Option 1: Netlify (Recommended)

1. Go to [netlify.com](https://netlify.com) and sign up for free
2. Click "Deploy manually" or "Sites" → "Deploy manually"
3. Drag and drop the entire `public` folder from this project
4. Your site will be live instantly!

### Option 2: GitHub Pages

1. Create a new repository on GitHub.com
2. Upload all files from this project to the repository
3. Go to Settings → Pages
4. Set source to "Deploy from a branch"
5. Select "main" branch and "/public" folder
6. Save and wait for deployment

## Features

- Responsive design for all devices
- Contact forms with email integration
- Package showcase
- Hall listings
- Gallery section
- WhatsApp booking integration

## Customization

- Replace `YOUR_EMAIL_HERE` in `public/index.html` with your actual email
- Update contact information in the contact section
- Add real photos to replace placeholder gallery items
